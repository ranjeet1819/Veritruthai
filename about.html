About TrueSource Project

TrueSource is an enterprise-grade multi-modal forensic architecture explicitly conceptualized to target and mitigate the algorithmic threats of digital manipulation, automated fake documentation, and highly structural deepfakes. Developed as a Capstone-II project at the Indian Institute of Technology Patna, this unified system addresses the critical challenge of verification in the age of pervasive Generative AI.

👁️ Vision Forensics

Driven by MTCNN face detection layers coupled with EfficientNet-V2 architectures. This module processes static media streams and long-form video frame samplings to detect high-frequency compression inconsistencies and facial artifact splicing anomalies with up to 96.4% precision.

✍️ NLP Text Evaluation

Powered by fine-tuned RoBERTa-base layers checking perplexity variations and length burstiness patterns inside unstructured articles. It automatically distinguishes natural human literary flows from AI-generated textual fragments.

⚡ MLOps Integration & Deployment

Built over an asynchronous FastAPI architecture capable of running concurrent non-blocking worker pools. The entire environment is containerized using Docker and securely hosted via live cloud environments for efficient scale and performance access logs.
